# To-Do App

## Project Overview
This To-Do App is a straightforward tool designed to help users effectively manage their daily tasks. Built with Node.js on the backend and EJS for the frontend, it supports basic CRUD operations (Create, Read, Update, Delete) for tasks.

## Features
- **Create Tasks**: Users can add new tasks with a title and optional due date.
- **View Tasks**: Displays all tasks in a list.
- **Update Tasks**: Users can edit details of existing tasks.
- **Delete Tasks**: Allows removal of tasks no longer needed.

## Tech Stack
- **Backend**: Node.js, Express
- **Frontend**: EJS, CSS
- **Database**: Choose between MongoDB or SQLite

## Getting Started

### Prerequisites
You will need Node.js and npm installed on your computer to run this project. To install these, follow the instructions from the [Node.js official website](https://nodejs.org/).

### Installation
Follow these steps to get your development environment set up:
- npm install
- DB_URI=mongodb://localhost:27017/tododb
- PORT=3000
